# Privacy Policy

Version: 1.0  
Effective Date: November 3, 2025

## Clear Statement

**We do not collect, store, process, or transmit any of your personal information.**

This is a simple and straightforward image stitching tool, and all operations are performed locally on your device.

## No Network Permission

This application **does not request any network permissions**, and the code **contains no network access related code**. We cannot and will not collect any of your information.

## No Data Collection

We do not care about any of your information, including but not limited to:
- Your name, phone number, email
- Your location information
- Your usage habits
- Content of images you process
- Your device information

**We do not collect anything.**

## Local Processing

All functions are executed entirely locally on your device:
- Image selection
- Image stitching
- Result saving

## Permission Information

The app supports multiple selectors, and **can be used even without granting any permissions**:

### 1. Storage Access Framework Selector (SAF)
- **No permissions required**
- Compatible with all SDK versions (29-36)
- Secure file access through system file picker

### 2. System Photo Picker (PhotoPicker)
- **No permissions required**
- Compatible with SDK 31+ or devices that can receive Google component updates
- Google-recommended modern selection method

### 3. Embedded Picker
- Requires storage permission: `READ_MEDIA_IMAGES` (SDK 33+) or `READ_EXTERNAL_STORAGE` (SDK 32-)
- Powerful selector extracted from the ImageToolbox project
- Other selectors can still be used even without granting permissions

## Stored Content

The app only stores necessary setting information locally:
- Theme preferences
- Stitching parameters
- Output settings

This information is stored entirely on your device and will not be uploaded anywhere.

## Absolute Transparency

You can check our source code at any time to confirm we do not have:
- Network access code
- Data collection code
- Any code that violates your privacy

**To reiterate: We are not interested in any of your information and have no ability to obtain it.**
