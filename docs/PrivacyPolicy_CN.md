# 隐私声明

版本：1.0  
生效日期：2025年11月3日

## 明确声明

**我们不收集、不存储、不处理、不传输您的任何个人信息。**

这是一个简单直接的图片拼接工具，所有操作均在您的设备本地完成。

## 无网络权限

本应用**不请求任何网络权限**，代码中**没有任何网络访问相关代码**。我们无法、也不会收集您的任何信息。

## 无数据收集

我们对您的任何信息都不关心，包括但不限于：
- 您的姓名、电话、邮箱
- 您的位置信息
- 您的使用习惯
- 您处理的图片内容
- 您的设备信息

**我们什么都不收集。**

## 本地处理

所有功能完全在您的设备本地执行：
- 图片选择
- 图片拼接
- 结果保存

## 权限说明

应用支持多种选择器，**即使不授予任何权限也能使用**：

### 1. 存储访问框架选择器 (SAF)
- **无需任何权限**
- 适用于所有 SDK 版本 (29-36)
- 通过系统文件选择器安全访问文件

### 2. 系统照片选择器 (PhotoPicker)
- **无需任何权限**
- 适用于 SDK 31+ 或能接收谷歌组件更新的设备
- 谷歌推荐的现代化选择方式

### 3. 嵌入式选择器 (Embedded Picker)
- 需要存储权限：`READ_MEDIA_IMAGES` (SDK 33+) 或 `READ_EXTERNAL_STORAGE` (SDK 32-)
- 从 ImageToolbox 项目提取的强大选择器
- 即使不授予权限也可使用其他选择器

## 存储内容

应用仅在本地存储必要的设置信息：
- 主题偏好
- 拼接参数
- 输出设置

这些信息完全存储在您的设备上，不会上传到任何地方。

您可以随时检查我们的源代码，确认我们没有：
- 网络访问代码
- 数据收集代码
- 任何侵犯您隐私的代码

**再次强调：我们对您的任何信息都不感兴趣，也无能力获取。**
