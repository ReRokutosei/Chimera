# 隐私政策

版本：1.1  
生效日期：2026年4月21日

## 1. 数据处理原则
Chimera 秉持**最小化数据处理**与**本地化处理**原则。本应用的核心设计目标是在不触及用户个人隐私的前提下提供工具服务。

## 2. 零数据收集
本应用不具备任何形式的远程数据收集、存储或传输功能。
- **个人信息**：我们不收集您的姓名、联系方式、位置或任何身份识别信息。
- **内容数据**：您处理的图片内容仅存在于设备内存及本地临时存储中，不会上传至任何服务器。
- **行为数据**：我们不监控您的使用习惯或点击行为。

## 3. 网络权限说明
本应用**未申请且不使用网络权限**（`INTERNET`）。从技术架构上杜绝了数据外泄的可能性。

## 4. 权限使用及透明度
应用申请的相关权限仅用于实现本地功能，用户可根据需求选择授予：
- **存储访问 (SAF/PhotoPicker)**：优先采用系统级安全选择器，在无需授予应用存储权限的情况下即可读取图片。
- **媒体读取权限**：仅在用户主动选择使用“嵌入式选择器”时申请，用于扫描本地媒体库。

## 5. 本地存储内容
本应用仅在设备本地存储必要的配置信息（如主题偏好、拼接参数等）。这些数据存储在应用的私有沙盒目录中，不会被其他应用访问。

## 6. 源代码透明度
作为开源项目，您可以随时审计我们的源代码以验证上述声明的真实性。

## 7. 联系与反馈
虽然本应用不收集隐私数据，但如果您对隐私相关事宜有任何疑问，可以通过 GitHub Issue 与我们联系。
